# FinTech SaaS Frontend — Capstone Project
A modern, responsive FinTech landing page built entirely using **HTML & CSS** as part of the Web Development Casptone Project.  
This project demonstrates clean UI design, layout structuring, semantic HTML, component-based styling, and polished presentation — all without using JavaScript.

## 📌 Project Overview
The project is a fully responsive landing page for a fictional FinTech product.  
It showcases smart payment handling, analytics, pricing tiers, testimonials, FAQs, and a clean contact form — all implemented with lightweight, accessible HTML & CSS.

This capstone project was created by **Prabhat Bhatia**.

## 🎯 Features

### ✅ **Modern Hero Section**
- Big headline with marketing-friendly layout  
- CTA buttons ("Start Free Trial", "View Demo")  
- Clean hero image with soft shadows

Lightweight emoji-based brand row (no external icon libraries).
